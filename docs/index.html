Steps:

  1. Install Node JS. Node itself is just an executable, so you donâ€™t really need to install it, but doing so will be easier and will install the node package manager (npm) as well.

  2. Install MySQL to persist data.

  3. Check-out project

    $ git clone https://github.com/teusemanuel/node-backend-server.git

  4. Checkout of project dependencies in command-line

    • go to node-backend-server checkout folder

      $ cd ~/folder-checkout

    • install dependencies

      $ npm install

    • in project root folder go to:

      $ cd node_modules/typings

    • install typings dependencies for working with typescript

      $ typings install
